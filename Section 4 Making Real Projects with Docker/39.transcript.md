   * To run the app locally:

     * Install dependencies: `npm install`
     * Start the server: `npm start`
   * Both commands assume `npm` is installed.

2. **Planning the Dockerfile**

   * Our Dockerfile will follow a pattern similar to what we did for Redis:

     1. Specify a base image.
     2. Run commands to install dependencies.
     3. Set a default command for when the container starts.

   * For Node.js, a first attempt might be:

     * Base image: `alpine`
     * Run `npm install` to install dependencies
     * CMD: `npm start`

3. **Creating the Dockerfile**

   * Open your code editor in the project directory.
   * Create a file named `Dockerfile` (capital D, no extension).
   * Add the following content:

     ```Dockerfile
     # Specify a base image
     FROM alpine

     # Install dependencies
     RUN npm install

     # Set default command
     CMD ["npm", "start"]
     ```

4. **Attempting to Build**

   * In the terminal, verify the `Dockerfile` exists.
   * Run:

     ```bash
     docker build .
     ```
